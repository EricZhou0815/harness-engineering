## Component Patterns

### Button

```
Primary:   bg=brand-primary, text=white, hover=brand-primary-dark
Secondary: bg=transparent, border=border-default, text=text-primary
Danger:    bg=feedback-error, text=white
Disabled:  opacity=0.4, cursor=not-allowed, no-hover-effects
```

Sizes: `sm` (32px height), `md` (40px height), `lg` (48px height)

### Input Field

```
Default:   border=border-default, bg=surface-base
Focus:     border=brand-primary, ring=brand-primary/20
Error:     border=feedback-error, ring=feedback-error/20
Disabled:  bg=surface-raised, opacity=0.6
```

### Cards

```
Surface:  bg=surface-raised, border-radius=8px, shadow=subtle-elevation
Padding:  16px (compact) | 24px (default) | 32px (spacious)
```

## Animation Tokens

```
Duration Fast:    100ms  (micro-interactions, toggles)
Duration Default: 200ms  (panel open/close, transitions)
Duration Slow:    400ms  (page transitions, large reveals)
Easing:           cubic-bezier(0.4, 0, 0.2, 1)   (standard material easing)
```

Avoid animation durations over 500ms for interactive elements.
Motion must respect the `prefers-reduced-motion` media query.
